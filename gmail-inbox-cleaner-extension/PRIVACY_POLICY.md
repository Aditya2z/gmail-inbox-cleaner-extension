Privacy Policy for Gmail Mark All As Read Extension

This Chrome extension does not collect, store, transmit, or share any personal or sensitive user data.

All functionality runs locally in the user's browser and only interacts with the Gmail webpage to automate marking emails as read.

The extension does not:
- Collect personal information
- Track browsing activity
- Send data to external servers
- Use analytics or tracking tools

Permissions such as activeTab, scripting, and host permissions are used solely to interact with the Gmail interface when the user triggers the extension.

If you have questions, please contact the developer.
